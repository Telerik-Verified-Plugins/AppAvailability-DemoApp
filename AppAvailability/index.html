AppAvailability Plugin Demo

The AppAvailability plugin allows your app to determine if a specific other app is available on the user's device.

It requires an URI Scheme (twitter://) on iOS or a Package Name (com.twitter.android) on Android.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.
